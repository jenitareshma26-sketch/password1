Password1 🔐

A simple and secure Password Generator application built with Python. This project helps users create strong, random passwords to improve online security and protect personal accounts from unauthorized access.

Features
Generate strong and secure passwords
Customizable password length
Includes:
Uppercase letters (A-Z)
Lowercase letters (a-z)
Numbers (0-9)
Special characters (!@#$%^&*)
User-friendly interface
Fast password generation
Helps improve account security
Technologies Used
Python
Tkinter (GUI)
Random Module
String Module

Installation
Clone the repository:
git clone https://github.com/jenitareshma26-sketch/password1.git
Navigate to the project directory:
cd password1
Install dependencies (if any):
pip install -r requirements.txt
Run the application:
python main.py
How It Works
Open the application.
Select the desired password length.
Click the Generate Password button.
A strong random password will be created instantly.
Copy and use the password for your accounts.
