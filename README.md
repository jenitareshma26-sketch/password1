# Password1 🔐

A simple and secure Password Generator application built with Python. This project helps users create strong, random passwords to improve online security and protect personal accounts from unauthorized access.

## Features

- Generate strong and secure passwords
- Customizable password length
- Includes:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!@#$%^&*)
- User-friendly interface
- Fast password generation
- Helps improve account security

## Technologies Used

- Python
- Tkinter (GUI)
- Random Module
- String Module

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/jenitareshma26-sketch/password1.git
```

### 2. Navigate to the project directory

```bash
cd password1
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python main.py
```

## How It Works

1. Open the application.
2. Select the desired password length.
3. Click the **Generate Password** button.
4. A strong random password will be created instantly.
5. Copy and use the password for your accounts.
